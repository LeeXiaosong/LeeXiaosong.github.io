Markdown Introduction（2）

Created2022-02-01|Updated2022-02-02
|Post View:|Comments:

Markdown is a Lightweight markup language, which represents the text format by adding symbols in front of the text without considering the text layout (using the same format), thus greatly simplifying the editing cost and improving efficiency

Markdown 是一种轻量级标记语言，通过在文本前增加符号来表示文本格式，不必考虑文本的排版问题（使用的是同一套格式），从而大大简化编辑成本提高效率

1 标题格式

使用#可以表示一到六级标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2 段落格式

Markdown的段落换行有两种

第一种是用两个（多个）以上的空格加回车换行
第二种是中间空一行表示换行

3 字体格式

Markdown的字体主要有一下几种

正体文本：不加任何符号
斜体文本1：*斜体*
_斜体文本2_：_斜体_
粗体文本1：**粗体**
__粗体文本2__：__粗体__
粗斜体文本1：***粗斜体***
___粗斜体文本2___：___粗斜体___

4 各种线

4.1 分隔线

可以在一行中用三个以上的星号或者底线来建立一个分隔线
***：

___：

4.2 下划线

下划线可以通过HTML的<u>标签来实现
以<u>为起始，以</u>为终止

5 文字颜色及大小设置

字体、字号以及颜色可在<font ……></font>中同时设置

<font face=”宋体” size=3 color=#fe346e >宋体 三号 红色:
宋体 三号 红色
<font face=”宋体” size=3 color=#ffd800 >楷体 三号 黄色:
楷体 三号 黄色
<font face=”微软雅黑” size=3 color=#1890ff >微软雅黑 三号 蓝色
微软雅黑 三号 蓝色

7.2 无序列表

无序列表用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容
