准备

1 目的

用Markdown实现跨平台笔记记录

2 方案

平台：Visual Studio Code + 坚果云

3 所需插件

3.1 实时预览：Paste Image

3.2 图片粘贴：Markdown All in One

为了和Windows使用习惯保持一致，在Ubuntu系统的键盘快捷键中设置如下

如果要将截图直接粘贴进文档，需要用到Paste Image插件

默认快捷键Ctrl+Alt+V进行粘贴，该插件可以把剪贴板上图片保存在指定的路径中同时将图片粘贴进md文档里。
Paste Image的相关参数设置如下

此处设置的是图片保存名格式：${currentFileNameWithoutExt}_Y-MM-DD-HH-mm-ss


此处设置图片的保存位置是当前文件下（相对路径）的Img_source文件夹中：${currentFileDir}/Img_source

3.3 文件导出：Markdown PDF


安装后在编辑界面点击鼠标右键即可导出相关格式

3.4 PDF格式预览：vscode-pdf
